# 💻 PASS 동아리 활동 기록 (C언어)
 
> **활동 기간:** 2026학년도 1학기  
> **기술 스택:** `C Language`

---

## 📌 프로젝트 소개
이번 학기 **PASS** 동아리에서 진행한 **C언어 기초 스터디 및 게임 제작 프로젝트**를 기록하는 저장소입니다.  
동아리장으로서 부원들과 함께 코딩의 기초를 다지고, 단계별 실습을 통해 하나의 완전한 프로그램을 완성해보는 경험을 가졌습니다. 처음에는 낯설고 막막했지만, 서로 소통하고 협력하며 함께 성장한 결과물들을 담고 있습니다.

---

## 🕹️ 주간 주요 활동 내용

### 1단계: C언어 기초 및 반복문 마스터 (`별 찍기`)
* **설명:** 코딩의 가장 기본이 되는 제어문과 반복문(for, while)의 개념을 확실히 다지기 위한 활동입니다.
* **주요 기능:** * 사용자 입력에 따른 다양한 형태의 별 기호(`*`) 출력 (직각삼각형, 역삼각형, 피라미드 등)
  * 다중 반복문(Nested Loop)의 흐름 이해 및 논리적 사고력 향상

### 2단계: 조건문과 무작위 수를 활용한 mini 게임 (`Up & Down 게임`)
* **설명:** 기초 개념을 바탕으로 변수, 조건문, 랜덤 함수를 종합적으로 활용해보는 첫 번째 게임 제작 활동입니다.
* **주요 기능:**
  * `rand()` 함수를 활용한 1~100 사이의 임의의 정수 생성
  * 사용자의 입력값과 정답을 비교하여 "Up" 또는 "Down" 힌트 제공
  * 기회 제한 및 입력 횟수 카운팅 기능 구현

### 3단계: 배열과 알고리즘의 심화 활용 (`2차원 배열 기반 성벽 디펜스 게임`)
* **설명:** 이번 학기 PASS 동아리 활동의 최종 결과물로, 2차원 배열을 활용해 맵을 구현하고 적의 침략으로부터 성벽을 지키는 턴제 디펜스 게임입니다.
* **주요 기능:**
  * `int map[ROW][COL]` 구조를 활용한 격자형 맵 및 성벽(Wall), 적(Enemy) 위치 표현
  * 턴 경과에 따른 적의 이동 알고리즘 구현 및 성벽 충돌 판정
  * 배열 내 데이터 갱신을 통한 게임 흐름 제어 능력 향상
